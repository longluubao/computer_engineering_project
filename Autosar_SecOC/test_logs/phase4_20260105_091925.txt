  ================================

  +----------+--------------------+-----------+----------------+
  |  Header  |   Authentic PDU    | Freshness | MAC (CMAC)     |
  | (opt. 1B)|    (variable)      |  (1-8B)   | (4 bytes)      |

  |<---------------------- Secured I-PDU ---------------------->|

  Example: 8-byte PDU with Classical Mode
  Total: 1 + 2 + 1 + 4 = 8 bytes

  +------+----------+---------+--------------------+

  POST-QUANTUM CRYPTOGRAPHY INTEGRATION
================================================================================

  Algorithms: ML-KEM-768 (FIPS 203) + ML-DSA-65 (FIPS 204)
  Security Level: NIST Category 3 (~192-bit classical equivalent)

  ML-KEM-768 (Key Encapsulation Mechanism):
  ==========================================

  Purpose: Quantum-resistant session key establishment
  Use Case: Secure key exchange between ECUs over Ethernet

  +-------------------+----------------+----------------------------------+
  | Parameter         | Size           | Description                      |
  +-------------------+----------------+----------------------------------+
  | Public Key        | 1,184 bytes    | Shared with peer                 |
  | Secret Key        | 2,400 bytes    | Kept private                     |
  | Ciphertext        | 1,088 bytes    | Encapsulated shared secret       |
  | Shared Secret     | 32 bytes       | Derived session key material     |
  +-------------------+----------------+----------------------------------+

  Key Exchange Flow:
  ------------------
    ECU A (Alice)                            ECU B (Bob)
         |                                        |
         |------ Public Key (1184 bytes) -------->|
         |                                        |
         |                            Encapsulate()|
         |                                        |
         |<----- Ciphertext (1088 bytes) ---------||
         |                                        |
    Decapsulate()                                 |
         |                                        |
    [Shared Secret]                    [Shared Secret]
       32 bytes                           32 bytes
         |                                        |
    HKDF Derivation                   HKDF Derivation
         |                                        |
    [Session Keys]                    [Session Keys]
      Enc: 32B                          Enc: 32B
      Auth: 32B                         Auth: 32B


  ML-DSA-65 (Digital Signature Algorithm):
  =========================================

  Purpose: Quantum-resistant message authentication
  Use Case: Sign every PDU for integrity and authenticity

  +-------------------+----------------+----------------------------------+
  | Parameter         | Size           | Description                      |
  +-------------------+----------------+----------------------------------+
  | Public Key        | 1,952 bytes    | Stored in ECU, used for verify   |
  | Secret Key        | 4,032 bytes    | Stored securely, used for sign   |
  | Signature         | 3,309 bytes    | Appended to Secured I-PDU        |
  +-------------------+----------------+----------------------------------+

  Signature Flow:
  ---------------
    Sender                                      Receiver
         |                                           |
    [AuthPdu]                                        |
         |                                           |
    ML-DSA-65 Sign()                                 |
         |                                           |
    [Secured I-PDU]                                  |
    = AuthPdu + Fresh + Sig (3309B)                  |
         |                                           |
         |-------- Ethernet Transmission ----------->|
         |                                           |
         |                                  ML-DSA-65 Verify()
         |                                           |
         |                                  PASS: Forward AuthPdu
         |                                  FAIL: Reject PDU
